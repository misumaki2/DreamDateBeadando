Weboldal röviden: Egy randi tervező weboldal aminek a 3 oldalán egy rövid játék van és a negyedik oldalán meg egy form. Ahhoz hogy a form látszódjon, rá kell kattintani az egyik fizetési plan-re.

Javascript kód: js/script.js

Kód röviden: A kód első felében a minigame van. Van egy "calculateCompatibility" function ami a nevek karaktereinek az unicode-jának alapján vissza ad egy szinte véletlen számot ami megmutatja, hogy "a két név mennyire szerelmes egymásba".
Utána van egy "handleLoveTest" nevű függyvény ami csak megszerzi a két névet az előző function-nel kiszámolja az esélyeket és utána kiírja és egy listába is kiírja, hogy látszódjanak az előzők is.
A kód második felében a form megjelenítése és validálása van.

Font: Google font (Montserrat)
